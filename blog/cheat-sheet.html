Strings

  • "bootcamp".capitalize = "Bootcamp"
  • "bootcamp".upcase = "BOOTCAMP"
  • "BOOTCAMP".downcase = "bootcamp"
  • "bootcamp".length = 8
  • "bootcamp".include(o) = true
  • "bootcamp".to_sym = :bootcamp
  • "bootcamp".center(10) = " bootcamp "
  • "bootcamp".delete "tc" = "booamp"
  • "bootcamp".insert(0, "dev") = "devbootcamp"

Operators

  • + => 5 + 7 = 12
  • - => 5 - 7 = -2
  • * => 5 * 7 = 35
  • / => 12 / 4 = 3
  • % => 12 % 4 = 0
  • ** => 2 ** 3 = 8
  • += => x = 0; x += 2; x = 2 => x = 4
  • > => 5 > 7 = false
  • < => 5 < 7 = true
  • >= => 5 >= 7 = false
  • <= => 5 <= 7 = true
  • == => 5 == 7 = false
  • != => 5 != 7 = true
  • && => 5 == 5 && 7 == 7 => true
  • || => 5 == 5 || 7 == 7 => true

Data Structures

Array

  • Array.new => Create a new array
  • numbers = Array.new([2, 0, 1, 5])
  • numbers.size = 4
  • numbers.length = 4
  • numbers[0] => 2
  • numbers.first => 2
  • numbers.last => 5
  • numbers.pop = 5 (new array [2, 0, 1])
  • numbers.push(6) = new array [2, 0, 1, 6]
  • numbers.join = "2016"
  • numbers.each = do |i| (function); end (each loops through each element)
  • numbers.map = creates a new array, does not destroy the original
  • numbers.map! = destroys original array

Hashes

  • {key => value} = creates a new hash
  • pet_names = {:cat => "Toby"}

Classes

  • class Dog...end = creates a new class called Dog
  • def initialize...end = object initialization
  • @name = name => creates an instance variable available throughout the class
  • @@name = class variable
  • $name => global variable
  • (*arguments) = allows for a variable number of arguments
  • self = calls the object itself
  • attr_reader: , attr_writer: , attr_accessor: variable => allows access to variables within a class

Reserved Words

  • def = definition
  • true = boolean
  • false = boolean
  • end = ends methods
  • if = if/else statement
  • elsif = if/else statement
  • else = if/else statement
  • for = starts for loops
  • class = creates a new class
  • return = prints content to the console
  • case = case loops
  • when = case loops
  • then = case loops
  • self = self-calling variable
